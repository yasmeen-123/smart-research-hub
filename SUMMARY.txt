                    UNDERSTANDING GIT CONCEPTS
════════════════════════════════════════════════════════════════════════

SAVED vs COMMITTED vs PUSHED:

1. SAVED (Ctrl+S in editor)
   • File is written to disk
   • Not tracked by Git yet
   Status: ❌ You DON'T have this

2. COMMITTED (git commit)
   • Changes saved to Git's local database
   • Tracked in version control
   Status: ✅ You HAVE this

3. PUSHED (git push)
   • Commits uploaded to GitHub
   • Available online and backed up
   Status: ✅ You HAVE this
